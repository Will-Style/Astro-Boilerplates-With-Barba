# Astro-Boilerplates


## Usage

### Project setup

Open `package.json`, and update properties.
```
"name": "Site name here.",
"description": "Site discription here.",
```

Installation
```
yarn install
```

### Run the following commands 

Compiles and hot-reloads for development
```
yarn dev
```

Compiles and minifies for production
```
yarn build
```
